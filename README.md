# 🔍 OSINT Recon Kit

웹 기반 OSINT(Open Source Intelligence) 정찰 도구 모음입니다. 도메인을 입력하면 다양한 정보 수집 도구로 즉시 연결됩니다.

## ✨ 주요 기능

### 🎯 Google Dorking (16개 도구)
- 문서 파일 검색 (PDF, DOC, XLS, PPT 등)
- 설정 파일 검색 (.env, config 등)
- 백업 파일 및 압축 파일 검색
- 관리자 페이지 및 자격증명 검색
- API Key 및 웹쉘 검색

### 🔍 정보 수집 (6개 도구)
- Wayback Machine - 과거 웹사이트 스냅샷
- robots.txt, phpinfo(), .git 디렉토리 확인
- Shodan - IoT 장치 검색
- Certificate Search - SSL 인증서 조회

### 🌍 서브도메인 & DNS (4개 도구)
- DNSDumpster, SecurityTrails
- VirusTotal, Censys

### 🔎 코드 & 리포지토리 (4개 도구)
- GitHub Code Search
- grep.app, PublicWWW
- Pastebin Search

### 🛡️ 취약점 & 익스플로잇 (4개 도구)
- Exploit Database
- CVE Details, NVD, Vulners

### 📊 분석 & 모니터링 (6개 도구)
- BuiltWith, Wappalyzer - 기술 스택 분석
- URLScan, SSL Labs
- Security Headers, Hardenize

## 🚀 사용방법

1. 웹 브라우저에서 `index.html` 파일 열기
2. 도메인 입력란에 조사할 도메인 입력 (예: example.com)
3. 원하는 도구 카드 클릭
4. 새 탭에서 자동으로 해당 도구 열림
